## ðŸ›  Installation & Set Up

1. Install the Gatsby CLI

   ```sh
   npm install -g gatsby-cli
   ```

2. Install and use the correct version of Node using [NVM](https://github.com/nvm-sh/nvm)

   ```sh
   nvm install
   ```

3. Install dependencies

   ```sh
   yarn
   ```

4. Start the development server

   ```sh
   npm start
   ```

## ðŸš€ Building and Running for Production

1. Generate a full static production build

   ```sh
   npm run build
   ```

1. Preview the site as it will appear once deployed

   ```sh
   npm run serve
   ```

## ðŸ›  Setup with Docker

1. Install Docker Engine and Docker-compose via [Docs](https://docs.docker.com/engine/install/)

2. Clone repo.
   
   ```sh
   git clone https://github.com/duyhenryer/me.git
   ```
3. Run Docker-compose

   ```sh
   docker-compose up -d
   ```
4. Preview the site on Website via localhost.

   ```sh
   http://localhost:8080
   ```
